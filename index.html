📦 NpmChecker

A tiny dashboard for the npm packages you care about. It pulls download history, release info, GitHub stars/PRs, and flags whether traffic looks like real use vs. bots/mirrors — all live in your browser. Nothing is sent anywhere; your list is stored locally on this device.

  • Paste package names below (one per line, or comma/space separated).
  • Scoped names work: @scope/name. Versions are ignored (lodash@4.17.21 → lodash).
  • Or upload a .txt file with one name per line.
  • Change the list anytime via the ⚙ Settings button.
