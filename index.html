📦 PackageChecker

A tiny dashboard for the packages you care about across npm, PyPI and crates.io. It pulls download history, releases, dependents and GitHub stars/PRs, and flags whether traffic looks like real use vs. bots/mirrors — all live in your browser. Nothing is sent anywhere; your list is stored locally on this device.

  • One per line (or comma/space separated). Prefix the ecosystem: pypi:requests, crates:serde. A bare name = npm.
  • Aliases: py:/pip: → PyPI, cargo:/rs: → crates.
  • Scoped npm works (@scope/name); versions are ignored.
  • Or upload a .txt. Change the list anytime via ⚙ Settings.
